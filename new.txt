Git Stash Commands – Notes
🔹 1. git stash
git stash

👉 Temporarily saves your uncommitted changes and cleans your working directory.

✅ Why use it?
When you are working on something but need to:
Switch branches
Pull latest code
Fix something urgent
💡 What happens?
Your changes are saved in a stash stack
Your project becomes clean (like last commit)

git stash apply
git stash apply
git stash apply
git stash show stash@{1}
git stash show stash@{1}
🔹 What it Does

👉 Displays a summary of changes stored in a specific stash.

stash@{1} → Refers to the second stash (older one)
Shows which files were changed
Does not show full code changes by default
